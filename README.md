# SC2002-TBCA: Turn-Based Combat Adventure

## Overview
A Java-based turn-based combat game that runs on the command line interface (CLI). Built using a Model-View-Controller (MVC) architecture and applied Object-Oriented Programming (OOP) concepts.

## Project Structure

```text
SC2002-TBCA/
├── src/                    # Source code files
│   ├── controller/         # Game logic and state management
│   ├── model/              # Data models (characters, items, levels, etc.)
│   ├── view/               # User interface and input handling
│   └── App.java            # Entry point of the application
└── README.md               # Project documentation
```

## Compilation and Running

Follow these steps to compile and run the game from the project root:

### 1. Compile the Source Code
Compile `App.java` from the `src` directory:

```bash
javac ./src/App.java
```

### 2. Run the Application
Execute the compiled `App` class:

```bash
java ./src/App
```
